About This Demo

This repository demonstrates common WCAG (Web Content Accessibility Guidelines) violations and their solutions. It includes:

  • Two HTML pages showing before and after accessibility fixes
  • Automated accessibility testing via GitHub Actions
  • Detailed documentation of each accessibility issue
  • Examples covering all four WCAG principles

Accessibility Features

Keyboard Navigation

This site is fully navigable using a keyboard:

  • Press Tab to move between interactive elements
  • Press Enter or Space to activate buttons and links
  • Press Shift+Tab to move to the previous element

A "Skip to main content" link appears when you first tab into the page.

Testing

This repository includes automated accessibility testing that runs on every push and pull request. The tests use axe-core to identify accessibility violations and generate detailed reports.

Created for educational purposes to promote web accessibility awareness.
